# comictose
A scraper that extracts some of the popular comic strips!

Visit this [link](https://manojkarthick.github.io/comictose/) to know more!

Here's the awesome list of Comics that are available!
* **Aunty Acid** 
* **Big Nate** 
* **Calvin and Hobbes** 
* **Cyanide and Happiness** 
* **Dilbert** 
* **Garfield**
* **Peanuts** 
* **SMBC** 
* **XKCD** 
* **ZenPencils**
